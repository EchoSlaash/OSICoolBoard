Ce programme permet d'organiser les données des logs générés par la CoolBoard dans un csv.
Les logs doivent tous être dans un seul dossier ne contenant pas d'autres fichiers.
Nécessite Java 8 ou supérieur.


##############################################################################################


Amélioration possible :

Demander à l'utilisateur les données qu'il souhaite récupérer.
Les clés utilisés dans le programme ne corresponde pas à celle des logs générés par la CB.
